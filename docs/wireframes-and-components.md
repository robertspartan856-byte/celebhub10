# CelebHub Wireframe-Ready Sections and Component Lists

## Purpose

This document breaks the MVP pages into wireframe-ready sections and reusable component inventories so design and frontend can work from the same page anatomy.

### Breakpoint guidance

- Mobile: 320 to 767 px
- Tablet: 768 to 1023 px
- Desktop: 1024 to 1439 px
- Large desktop: 1440 px and above

### Wireframe handoff checklist

1. Mark above-the-fold content on Home, Events Catalog, and Event Detail.
2. Show loading, empty, and error states for search and purchase flows.
3. Include success and validation states for newsletter, auth, and checkout forms.
4. Annotate sticky actions on mobile for buy and checkout flows.
